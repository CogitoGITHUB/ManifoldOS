* Manifold OS — Master Plan
:PROPERTIES:
:CREATED: :END:

I'm building Manifold as a complete rethinking of how a Guix-based system should work. 
No channels. No umbrella modules. Clean, intentional structure.

**Core Philosophy**
- The filesystem is the source of truth, not channels
- Every declaration must be clean and focused
- No more `(use-modules (giant-umbrella-module))`
- The system should feel like it belongs to me completely

* TODO Phase 1 — Change How Guix Itself Works
** Goal
Modify Guix's core so it no longer depends on channels for my own code.

** Tasks
- [ ] Study how Guix loads modules and resolves packages
- [ ] Understand the internal module loader and `GUIX_PACKAGE_PATH` limitations
- [ ] Create a modified Guix that uses my custom loader
- [ ] Test that my system can boot using only my declarations
