# Empiricist — Scope

## Authority Over

**Directories:**
- `/ManifoldOS/council/troubleshooting/` — diagnostic docs and solutions
- `/ManifoldOS/forge/` — experimental modules (testing breakage scenarios)

**Files:**
- Any file during active troubleshooting session
- Build logs (temporary analysis)
- Error outputs

**Domains:**
- Build failures
- Reconfigure errors
- Runtime breakage
- Package conflicts
- Service startup failures
- Performance issues

## Not Authority
- Architecture changes (structuralist)
- Package additions (curator)
- Documentation writing (propagandist)
- Reference maintenance (cartographer)

## Temporary Authority
During active troubleshooting, empiricist may temporarily modify:
- Any `.scm` file to test hypotheses
- Constitution.scm to isolate issues
- Any module to add debugging output

Must restore original state after diagnosis unless structuralist/curator approve permanent change.
